# Code of Conduct

This project follows the Contributor Covenant v2.1 to foster an open, welcoming, and harassment-free community.

Expected behavior

- Be respectful and considerate in all interactions.
- Use inclusive language and avoid targeted harassment.
- Assume good faith when proposing changes or commenting on work.

Unacceptable behavior

- Violence, harassment, or hateful language.
- Targeted insults, derogatory language, or discrimination.
- Public shaming or doxxing.
